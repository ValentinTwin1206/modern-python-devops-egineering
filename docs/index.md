# Modern Python Engineering

## Introduction

The practical course ***Modern Python Engineering*** deals with tools,
standards, and delivery workflows behind modern Python development.
Chapter 01 introduces the basics of Python environments, Chapter 02
focuses on Python packaging, and Chapter 03 provides a deep dive into
*uv* as a modern Python project and package manager. Chapter 04 then
brings that foundation together in a real-world Python application,
where you build a plugin for a CLI framework based on a GitHub template
repository while learning common design patterns, development
workflows, and CI/CD practices, as well as integrations with platforms
such as *GitHub*, *PyPI*, and *Docker Hub*.
